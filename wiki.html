Wiki du Projet : Breizh Hardware

Projet SAÉ 203 : Développement de solutions informatiques pour l'entreprise


1. Architecture du projet

Notre infrastructure repose sur un serveur web Apache hébergeant deux plateformes distinctes, pensées pour répondre à des besoins techniques ou autres :

  • Le Site Vitrine (WordPress) : C'est la vitrine publique de Breizh Hardware. Ce système de managment a été configuré avec un thème sombre pour refléter notre identité technologique. Il présente notre savoir-faire "pur beurre, pur processeur", l'histoire de notre fondation en 2025, ainsi que nos partenaires.

  • L'Intranet (Développement PHP/Bootstrap) : Un portail privé développé sans base de données SQL. L'architecture logicielle s'appuie exclusivement sur la manipulation de fichiers JSON pour le stockage des données (utilisateurs, clients, commandes).

2. Description et utilisation des fonctionnalités

🌐 Fonctionnalités de la Vitrine (Publique)

  • Présentation : Pages informatives détaillant nos valeurs, notamment le "Reconditionnement de matériels informatiques".
  • Module Partenaires : Affichage de nos collaborateurs externes.

🔒 Fonctionnalités de l'Intranet (Privé)

  • Authentification sécurisée : Portail de connexion avec gestion des sessions PHP et hachage des mots de passe.
  • Tableau de bord : Page d'accueil qui montre les données clés de l'entreprise (nombre de membres, nombre de clients).
  • Annuaire de l'entreprise : Liste interactive des membres de l'équipe avec photo de profil, fonction, et biographie. Intègre une barre de recherche en temps réel.
  • Annuaire Client : Base de données recensant nos clients avec des indicateurs visuels (ID, statuts Particulier/Entreprise).
  • Espace Administration : Module réservé aux administrateurs permettant la gestion complète du personnel (ajout de nouveaux collaborateurs via formulaire complet et suppression de fiches existantes).
